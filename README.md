# day-planner

## Description
This app allows you to keep track of your day hour by hour. Allows you to save notes.

## Installation/usage
Live Site: https://cojo12.github.io/day-planner/

1. Click link above
2. Click inside desired timeblock
3. Enter your notes
4. Click 'Save'
